Data Science Analysis

Analisis prediktif menggunakan machine learning untuk mengidentifikasi faktor dominan pembentuk loyalitas pelanggan restoran.

Data Science: Analisis Tingkat Kepuasan Restoran
Python • Machine Learning • EDA • Scikit-Learn • Data Visualization
https://images.unsplash.com/photo-1551288049-bebda4e38f71?auto=format&fit=crop&q=80&w=1200

Konteks Proyek

Proyek Analisis Tingkat Kepuasan Restoran berbasis Data Science, berfokus pada analisis prediktif untuk mengidentifikasi faktor dominan yang berfungsi sebagai tipping point dalam mengubah pelanggan dari status tidak puas menjadi puas dan loyal.

Cakupan Teknis

  • Melakukan pra-pemrosesan dataset (349 baris, 11 kolom), yang mencakup penanganan missing value, konversi data kategorikal menjadi format numerik biner, dan standardisasi fitur numerik menggunakan StandardScaler.
  • Menjalankan Exploratory Data Analysis (EDA) melalui visualisasi histogram dan correlation heatmap untuk memetakan distribusi penilaian pelanggan dan korelasi antar variabel.
  • Membangun dan menguji model klasifikasi machine learning (Logistic Regression, SVM Linear, dan Random Forest) untuk memprediksi status kepuasan pelanggan.
  • Mencapai kinerja model yang sempurna dengan tingkat Akurasi, Precision, Recall, dan F1-Score sebesar 1.00 pada pengujian data.
  • Menganalisis Feature Importance untuk menghasilkan rekomendasi bisnis strategis, yang membuktikan bahwa fasilitas tambahan (seperti pet friendly, area parkir, opsi vegan) dan persepsi harga (worth the price) adalah penentu utama loyalitas pelanggan.

Java MVC Application

Aplikasi desktop berbasis Java OOP dengan arsitektur MVC dan konektivitas database MySQL.

Java Desktop Application with MVC
Java (Swing/JavaFX) • OOP • MVC Pattern • MySQL
https://images.unsplash.com/photo-1555066931-4365d14bab8c?auto=format&fit=crop&q=80&w=1200

Deskripsi Proyek

Mengembangkan aplikasi desktop terintegrasi berbasis Java ("RocketFly") untuk mendigitalisasi operasional bisnis penyewaan motor dari pencatatan manual. Sistem ini dirancang dengan memisahkan logika bisnis, presentasi UI, dan manajemen data secara ketat menggunakan pola arsitektur Model-View-Controller (MVC).

Fitur & Implementasi

  • Menerapkan prinsip Object-Oriented Programming (OOP) dan merancang pemodelan sistem (Use Case, ERD) sebelum implementasi. Arsitektur codebase dikelola menggunakan pola MVC untuk memastikan skalabilitas dan kemudahan pemeliharaan.
  • Mengintegrasikan aplikasi dengan basis data relasional MySQL menggunakan koneksi JDBC. Mengimplementasikan sistem autentikasi role-based access control (Admin & Karyawan) untuk membatasi hak akses pengelolaan data operasional.
  • Membangun antarmuka interaktif menggunakan Java Swing. Mengembangkan logika otomatisasi untuk kalkulasi biaya sewa dinamis (per jam/hari) dan sinkronisasi status ketersediaan motor secara real-time saat transaksi terjadi.
  • Mengimplementasikan pustaka iReport/JasperReports untuk pembuatan struk dan laporan transaksi harian dalam format PDF. Mengintegrasikan JavaMail API untuk fitur pengiriman bukti transaksi langsung ke email pelanggan.
  • Melakukan pengujian unit (Unit Testing) menggunakan framework JUnit pada modul controller (CRUD motor, pelanggan, transaksi, dan user) untuk memvalidasi fungsionalitas dan keandalan sistem sebelum deployment.
